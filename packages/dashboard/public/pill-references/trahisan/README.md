# Trahisan (Трахисан) — Demo Pill Reference

Альтернативная безопасная демо-таблетка для хакатон-показа: **Trahisan / Трахисан**, производства **Engelhard Arzneimittel** (Германия). Пастилки для рассасывания при боли в горле, безрецептурные.

## Состав препарата (1 пастилка)

| Компонент | Доза |
|-----------|------|
| Хлоргексидина диглюконат | 1 мг |
| Тиротрицин                | 0.5 мг |
| Лидокаина гидрохлорид     | 1 мг |

## Внешний вид

- **Коробка**: белая с красными надписями "TRACHISAN" / "Трахисан", синие акценты, логотип Engelhard
- **Блистер**: 10 пастилок × 2 ряда (или 20 в одном ряду)
- **Пастилка**: круглая, плоская, белая или светло-кремовая, ~12-14 мм диаметр

## Что положить в эту папку

Аналогично Аскорутину — минимум **5 фото в разных ракурсах**:

| Файл | Описание |
|------|----------|
| `box-front.jpg` | Коробка лицом к камере (видно "TRAHISAN") |
| `box-back.jpg` | Обратная сторона коробки |
| `box-side.jpg` | Сбоку |
| `blister.jpg` | Блистер 20 пастилок |
| `pill-single.jpg` | Одна пастилка в ладони |
| `pill-multiple.jpg` | (опционально) несколько вместе |

**Желательно**: 1280×720+, разное освещение, разные углы.

## Где купить в Узбекистане

Любая аптека, ~10-25 K сум за пачку 20 шт. Безопасен для глотания на демо
(но рассчитан на рассасывание — для VOT демо демонстратор может просто
прожевать одну).

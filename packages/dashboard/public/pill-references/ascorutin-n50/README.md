# Аскорутин N50 — Demo Pill Reference

Демо-таблетка для хакатон-показа: **Аскорутин (Ascorutin)** производства ЛЕКХИМ (Украина), упаковка по 50 таблеток.

## Состав препарата (1 таблетка)

| Компонент | Доза |
|-----------|------|
| Аскорбиновая кислота (Витамин C) | 50 мг |
| Рутозида тригидрат (Рутин)       | 50 мг |

Безрецептурный витаминный препарат, укрепляет капилляры. Безопасен для глотания на демо.

## Внешний вид

- **Коробка**: белая с золотой рамкой, логотип "ЛЕКХИМ" + "АСКОРУТИН" крупными буквами, бок: "50 ТАБЛЕТОК"
- **Блистер**: внутри обычно 10 таблеток × 5 рядов (или 25×2)
- **Таблетка**: круглая, желтовато-зелёная (цвет рутина), плоская, без риски, диаметр ~7-8 мм

## Как использовать на демо

1. Реальная коробка Аскорутина у тебя в руке
2. На шаге `show_box` AI видит коробку → YOLO должен задетектить как `ascorutin_box`
3. На шаге `show_pills` достаёшь 1 таблетку → YOLO детектит как `ascorutin_pill`
4. На шаге `pill_closeup` Vision LLM (Qwen-VL) подтверждает: "вижу круглую жёлто-зелёную таблетку, форма соответствует Аскорутину"
5. Глотаешь → mouth_check видит пустой рот

## Тренировка YOLO на Аскорутине

После добавления фото:

```bash
# 1. Auto-annotate через Vision LLM (как мы делали для TB pills)
pnpm annotate:ascorutin

# 2. Train YOLO v2 с новым классом
pnpm train:yolo:ascorutin

# 3. Deploy на vast.ai (port 8004)
pnpm deploy:yolo
```

## Заметки для презентации

✅ Безопасно глотать (витамины)
✅ Доступен в любой аптеке Узбекистана (~5,000-15,000 сум)
✅ Маленький размер — легко спрятать/вернуть из руки если демо собьётся
✅ Жёлто-зелёный цвет — отличается от белой стены/ладони, легко детектится
⚠ НЕ путать с реальным TB-протоколом — на pitch чётко скажем "demo placeholder"
